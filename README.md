# 📖 About Project

**Note App** is a lightweight and beginner-friendly note management application developed using Python

The goal of this project is to provide a simple environment where users can:

* Create notes
* Save information
* Manage text data
* Practice working with files and Python logic

This project is small, clean, and easy to understand, making it perfect for beginners who want to learn practical Python programming concepts
